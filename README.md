# You_Jump_I_Jump  
相关博客：https://blog.csdn.net/ffcjjhv/article/details/85140212  
## 运行 
需要安装Tomcat后以类似 `http://localhost:8080/zjgame/zjgame.html `的方式访问，否则本地图片无法加载。  
注：本项目基于Three.js，是对微信小游戏跳一跳的html版改写，只供研究学习使用。
